07 - Functions

Use Functions to automate repetitive statements or group together a single task. Make sure to keep your console open while viewing the page (Ctrl + Shift + I)

# What are functions?

* Functions help group together a group of statements and call them all together.
* The group of statements usually have a common goal, i.e. a common function to serve

Different types of functions,

  • Using function keyword
  • Function Expression
  • Arrow Functions
  • IIFE (Immediately Invoked Function Expression)

# Using `function` keyword

function sayMyName() {
console.log("Hii, I'm Kunal");
}
sayMyName();

function giveMeANum() {
return 10;
}
const myNum = giveMeANum();
console.log(myNum);

# Function Expression

const myFunc = function () {
console.log("I'm in a function expression!");
}
myFunc();

const anotherFunc = myFunc;
anotherFunc();

# Arrow Functions

const arrowFunc = () => {
console.log('Arrow functions are cool!');
}
arrowFunc();

# IIFE (Immediately Invoked Function Expression)

(function () {
console.log('This is an IIFE');
})();

(() => {
console.log('This is another IIFE');
})();
