01 - Basic Console

Console a simple line using the console statement. Make sure to keep your console open while viewing the page (Ctrl + Shift + I)

# Statements

A statement is a line of instruction that does a certain activity.

console.log("Hello, World!");

In the above example we're using the `console.log` statement to print something to the browser console or the terminal depending on where it's being run.
