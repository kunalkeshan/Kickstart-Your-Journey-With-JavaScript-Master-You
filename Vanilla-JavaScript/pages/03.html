# What are comments?

Comments are lines of code that is ignored by the interpreter during the run time.

Comments can be written in two ways in JavaScript, single line comments and multi line comments.
- Single Line Comments - uses `//` and the comment text after the double forward slash.
- Multi Line Comments - uses `/* */` and the comment text in between the asterisks.

// this line is called a single line comment

// you can even make multi line comments using /* */

/**
* Just like this!
*/
